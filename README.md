# 4ASCEND

### 项目背景

基于五子棋改编的双人策略棋类游戏，黑方、白方轮流进行。
通过形成四连（ASCEND）对敌人造成伤害，利用魔力植物和反击机制扭转战局。

该游戏由橋野みずは设计并开发，本项目仅仅是对其的粗略模仿，在网站 https://unityroom.com/games/4ascend 可以游玩原版。

由于我的技术不足等原因，本项目几乎没有进行动画的还原，同时人机对战中的 ai 设计也十分粗糙。为此本项目提供了接入自定义 ai 的方法，可以使用你自己构思的程序来双方互相对战。

### 核心算法

Minimax 搜索：使用 Minimax 算法构建博弈树，模拟双方交替落子的过程，通过深度优先搜索评估局面。

Alpha-Beta 剪枝：在 Minimax 基础上引入 α-β 剪枝，大幅减少搜索分支，使 AI 能在合理时间内搜索更深层次，并随场面复杂程度自适应深度以保证效率。

移动排序（Move Ordering）：对候选步按启发式值排序（植物价值、中心控制、防守威胁），进一步优化剪枝效率，加速搜索。

深拷贝与快照恢复：使用 `copy.deepcopy` 保存完整的游戏状态（棋盘、血量、植物、待结算信息等），模拟落子后可无副作用地恢复到原始状态，这是搜索算法中回溯操作的关键实现。

递归回溯：在 AI 模拟中，对每个候选步进行“试走→评估→撤销”，体现了经典的回溯思想。
